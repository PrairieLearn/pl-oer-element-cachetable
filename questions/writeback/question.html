Example Cache Table

This example sets the cache to a static table using the is-material attribute. It also illustrates the show-dirty attribute for simulating writeback caches.

Given a direct-mapped cache with {{params.sets}} sets of {{params.block_size}}-byte blocks and a main memory that has {{params.addr_bits}}-bit addresses.

A load instruction is trying to load from address {{correct_answers.cache_access.0.address}}. If there is a cache hit, determine what data would be retrieved from the cache (be sure to add the 0x prefix). If there is a cache miss, determine whether a write-back would happen. If there would be a writeback type "writeback". If there would NOT be a writeback type "replace".
